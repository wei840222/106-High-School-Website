計算機程式設計課程

學校相關單位授予合格結業學生先修課程學分，使得修課學生得以抵免未來大學之修課學分。

教學目標與教材大綱構想

在基礎課程「計算機程式設計」中，依據合作高中職學校資訊科以及資訊相關課程或社團在程式設計方面的需求，將以C++ 為主要語言，教學目標為奠定學生在程式設計基礎能力，並參考美國Advanced Placement Program – Computer Science (AP-CS) 大學先修課程的概念，鼓勵參與程式設計檢測，加強學生程式設計相關概念並應用在解決問題的基本能力。主要教學目標為：

  • 培養學生分析問題，設計方法，並解決問題的能力。
  • 培養學生使用或實作常見演算法的能力。
  • 培養學生使用或實作常見資料結構的能力。
  • 培養學生設計C++ 物件導向程式的能力。

在課程教材大綱的構想上，「計算機程式設計」課程將依據大學資訊系程式設計課程的教材為基礎，從基本的觀念開始介紹，搭配程式設計檢測相關題型融入課堂及課後作業之練習，並依據個別學生程度之差異，適時進行內容與實施方式之調整。
