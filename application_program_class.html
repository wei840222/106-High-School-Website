依據合作高中職的現況及需求，進行專題製作指導

計算機程式應用課程

學校相關單位授予合格結業學生先修課程學分，使得修課學生得以抵免未來大學之修課學分。

教學目標與教材大綱構想

在實務應用課程「計算機程式應用」中，考量各校學生的興趣廣泛，開發應用軟體的經驗不一，為了能有效增進學習效果，預計將分成多組別，分別以支援課程教授專業或現今手機App，以及機器人等普遍受到重視的主題為例，透過如 Google App Inventor for Android 等工具，讓學生能依據興趣實際演練程式設計技能，並讓學生發揮創意，設計各種應用情境。目標在於透過實務應用的設計開發，輔導學生能多參與相關軟體專題競賽。為了增進主題分組教學的效果，學生分組將初步依據公平的篩選機制，如：程式基本能力，工具使用經驗等，平均分配名額給各高中職，若有多餘名額，則再留給其餘需求較高的學校，以便顧及各高中職校的需求。

在「計算機程式應用」實務課程方面，將依個別主題請不同研究領域老師介紹先備知識，並請研究生分享相關工具的使用，最後安排同儕相互觀摩，以便增進廣泛的了解。主要教材大綱構想為：

  • 物件導向程式觀念
  • 系統開發流程
  • 程式分析、測試、偵錯及實作
  • 標準資料結構：string, list, array
  • 基本演算法：sorting, searching

分組授課老師主題及組員限制

課程資訊 (已經在107/3/6開放報名)

先舉行一次共同課程 (日期未定)，然後由各專題負責老師與學生協調後續進行時間與地點，最後在學期結束前 (大約六月份) 會與上學期Intel專題的組別一起進行成果發表會。

北科大電子系 蕭榮修 老師

  • 主題方向：情境感知應用系統
  • 簡要描述：以嵌入式平台或行動裝置，結合無線通訊及感測技術，研發情境感知應用系統。
  • 基本條件：對 C/C++, Python程式語言、無線通訊及感測器技術有興趣。
  • 學生人數：2~3名學生一組, 最多1組。
  • 進行方式：晚間或周末 meeting，其餘時間由實驗室TA個別指導實作。
  • 時間地點：臺北科技大學電子系會議室及綜科館107實驗室。

北科大互動系 王聖銘 老師

  • 主題方向：情緒感知機器人之設計思考
  • 簡要描述：利用設計思考之方法，討論情緒機器人之服務情境，並整合表情偵測及語意分析演算法，建構情緒機器人之雛型系統。
  • 基本條件：數理能力優，具程式設計經驗，並願意接受跨領域的挑戰。
  • 學生人數：兩組（兩組最多6人）。
  • 進行方式：透過工作坊先行了解目前機器人之服務模式，而後上網自主查閱表情辨識及語意分析之開放原始碼之內容。利用Maker實作的方法，進行雛型系統開發，並作易用性評估。
  • 時間地點：星期一晚上1800-2100或星期六早上0930-1230

北科大資工系 郭忠義 老師

  • 主題方向：動態網頁程式設計
  • 簡要描述：以 Python 開發網頁應用系統。
  • 基本條件：基本程式設計、網路、HTML概念。
  • 學生人數：2~3名學生一組，最多6名。
  • 進行方式：實際上機寫程式。
  • 時間地點：臺北科技大學資工系，週一~週五晚上 18:00~20:00 選一天。

北科大資工系 江佩穎 老師

  • 主題方向：基礎電腦圖學
  • 簡要描述：電腦圖學基本概念及介紹, 並實際開發OpenGL程式。
  • 基本條件：熟悉C++程式語言。
  • 學生人數：一人一組。
  • 進行方式：授課以及討論實作方式。
  • 時間地點：晚間或週末在系上研討室進行。

北科大資工系 王正豪 老師

  • 主題方向：網路開放資料分析
  • 簡要描述：利用開放原始碼工具進行資料內容檢索及分析。
  • 基本條件：熟悉任何一種程式語言。
  • 學生人數：兩人一組，最多兩組。
  • 進行方式：先熟悉開放原始碼工具及開放資料，再進行系統開發，課程主要以討論方式進行。
  • 時間地點：預計於晚間或周末在臺北科技大學資工系進行。
